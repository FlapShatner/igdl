# Instagram Downloader Extension

This Chrome extension adds a download button to Instagram posts, stories, and reels, allowing you to easily download images and videos directly from Instagram.

## Features

- Download images and videos from Instagram posts.
- Download Instagram stories.
- Download Instagram reels.
- Simple download button integrated into the Instagram UI.

## Installation

1.  **Download the code:** Clone or download the ZIP file of this repository to your local machine. If you downloaded a ZIP file, unzip it.
2.  **Enable Developer Mode:**
    *   Open Google Chrome.
    *   Navigate to `chrome://extensions/`.
    *   In the top right corner, toggle the "Developer mode" switch to the on position.
3.  **Load the Extension:**
    *   Click the "Load unpacked" button that appears after enabling Developer mode.
    *   Navigate to the directory where you cloned or unzipped the repository files.
    *   Select the folder containing the extension's code (e.g., the root folder of the downloaded repository).
    *   Click "Select Folder".

The extension should now be installed and active. You should see its icon (if applicable) in your Chrome toolbar, and download buttons should appear on Instagram content.

## Usage

Once installed, navigate to Instagram. You will find download buttons added to posts, stories, and reels. Click the button to download the desired media.
